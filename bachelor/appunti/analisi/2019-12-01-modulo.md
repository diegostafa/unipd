### **Proprietà archimedea**

**ipotesi**:
* m è un qualsiasi numero positivo
* a è un qualsiasi numero positivo

**tesi**:
* ∃ n ∈ N : a*n > m

(utile per dimostrazioni successive)

#### **Proprietà del modulo**

**ipotesi**:
* a<sup>2</sup> ≤ b<sup>2</sup>

**tesi**:
* \|a\| ≤ \|b\|

NB: vale anche l'inversa
<br>
#### **Proprietà del modulo**

**ipotesi**:
* \|x\| ≤ a

**tesi**:
* -a ≤ x ≤ a

NB: vale anche l'inversa

**problem**

defined as:
* initial state
* successor function --> action to state unction
* goal test --> state to boolean function
* step cost --> the cost of the action

**solution generation**

to find the solution of a problem there are multiple search techniques:
* uninformed
    * breadth first
    * depth first
    * iterative deepening
    * bidirectional
    * deep limited
* informed
    * best first
    * A*
    * euristic
    * simulated annealing
    
**uninformed search**

from the initial state generates a tree of all possible paths, pick the one that cost less
